# About
Overlay makeup and 3D models onto a video stream using mediapipe, opencv, and pyvirtualcam.

# Setup
```
git clone [repo url]
cd virtual_makeup
uv venv venv
source venv/bin/activate
uv pip install opencv_python mediapipe pyvirtualcam
python test.py
```

Use instructions on the info box from the video stream output window.
